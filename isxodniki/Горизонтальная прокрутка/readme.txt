Использование
Хорошо, как можно использовать CHScrollListBox класса? Вот некоторые шаги, чтобы взять, вроде ручного труда, что будет вам там:

В редакторе ресурсов, создать шаблон диалога, который пройдет управления ListBox.

Укажите свойства списка в соответствии с рисунком выше (по крайней мере, горизонтальная прокрутка, выбор и рисование владельцем должны совпадать).

В заголовке файла класса вашего диалога, связанных с шаблоном, добавить:
# Включить  "HScrollListBox.h"

Определить переменную-член CHScrollListBox, вот так:
CHScrollListBox m_listBox ;

В файле реализации, в функции DoDataExchange (), добавьте строку:
DDX_Control ( PDX , IDC_YOUR_LISTBOX_CTRL_ID , m_listBox );

Этот вызов в свою очередь вызывает SubclassDlgItem (), которая делает трюк маршрутизации всех сообщений, предназначенных для управления нашим карту сообщений.
Конечно, IDC_YOUR_LISTBOX_CTRL_ID и m_listBox являются orbitrary имен, вы должны использовать свои собственные, а не.
